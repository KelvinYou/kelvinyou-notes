## Quick start

```shell
npm install --save-dev jest babel-jest
```

```json
/* Add to package.json */
"scripts": {
  "test": "jest"
}
```

```shell
# Run your tests
npm test -- --watch
```

- See: [Getting started](https://jestjs.io/docs/getting-started)

## Writing tests

```js
describe('My work', () => {
  test('works', () => {
    expect(2).toEqual(2)
  })
})
```
